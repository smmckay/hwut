Quick Start
===========

This section is designed to get the user up and running as quickly as 
possible. First the installation and its requirements are discussed. Second
the very basic concepts of deterministic tests are introduced. An example
shows practically how to use hwut for a software test. After this little 
promenade in the world of HWUT, the reader shall be familiar with its basic
ideas and ready to start his first software test.
